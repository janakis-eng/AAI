 Flower Image Classification (CNN)
 Description

This project uses a Convolutional Neural Network (CNN) to classify flower images into different categories such as rose, sunflower, tulip, daisy, and dandelion.

⚙️ Tools & Technologies
Python
    TensorFlow / Keras
    NumPy
    Matplotlib
    Streamlit
 Project Files
    classification.ipynb – Model training
    cnn_model.h5 – Saved model
    app.py – Streamlit app
    dataset/ – Flower images
How to Run
    Install requirements
    pip install -r requirements.txt
    Run the app
    streamlit run app.py
Output

    Upload a flower image and the model will predict its category.

 Note

    This is a basic deep learning project for image classification using CNN.
